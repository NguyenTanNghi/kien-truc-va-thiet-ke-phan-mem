Vì sao chọn Strategy?

Bạn có nhiều cách tính thuế (VAT, thuế tiêu thụ, thuế đặc biệt...) → đó là nhiều thuật toán khác nhau.

Strategy cho phép đổi thuật toán tính thuế linh hoạt theo từng sản phẩm mà không cần if/else dài.

Dễ mở rộng: thêm loại thuế mới chỉ cần thêm 1 class mới, không sửa code cũ nhiều.
